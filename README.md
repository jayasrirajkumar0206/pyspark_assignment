# PySpark Assignment

This repository contains PySpark practice questions covering real-time Data Engineering concepts.

## Questions Covered

### Question 1 – Customer Purchase Analysis

* Create DataFrames with custom schema
* Identify customers who bought only *iphone13*
* Find customers who upgraded from *iphone13* to *iphone14*
* Find customers who purchased all products

### Question 2 – Credit Card Masking & Partitioning

* Create DataFrame using different methods
* Manage partitions (increase and decrease)
* Mask card numbers using UDF
* Generate masked output

### Question 3 – User Activity Analysis

* Custom schema using StructType
* Dynamic column renaming
* Calculate user actions in the last 7 days
* Convert timestamp to date
* Write CSV and managed table

### Question 4 – JSON Processing & ETL

* Read nested JSON dynamically
* Flatten the DataFrame
* Compare explode functions
* Convert camel case to snake case
* Add load date and partition columns
* Write partitioned table

### Question 5 – Employee Data Pipeline

* Create DataFrames dynamically
* Salary aggregation
* Join and enrichment
* Bonus calculation
* External table creation
